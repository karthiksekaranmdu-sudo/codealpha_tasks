# E-COMMERCE STORE
Elite Shop is a simple and responsive e-commerce web application developed using HTML, CSS, and JavaScript. This project demonstrates the core functionalities of an online shopping platform, including user authentication, product browsing, shopping cart management, and order processing.

The application begins with a secure login page that verifies user credentials before granting access to the store. Users can explore various products, view product details, add items to their shopping cart, remove unwanted products, and place orders seamlessly.

Upon successful order placement, the application redirects users to an order confirmation page, providing a complete shopping experience.





## Features

🔐 User Login Authentication
Validates user credentials before allowing access to the store.

🛍️ Product Listing
Displays products with images, descriptions, and prices.

📄 Product Details
Allows users to view detailed information about each product.

🛒 Shopping Cart
Add products to the cart.
View all selected products.
Remove products from the cart.
Automatically calculates the total price.

📦 Order Processing
Enables users to place orders directly from the cart.

✅ Order Confirmation
Displays a success page after the order is placed successfully.

📱 Responsive Design
Optimized for desktops, laptops, tablets, and mobile devices.


## TECHNOLOGIES USED
HTML5 – Structure and content of the website.

CSS3 – Styling and responsive design.

JavaScript (ES6) – Dynamic functionality and user interactions.
